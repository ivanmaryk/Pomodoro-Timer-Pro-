🍅 Pomodoro Timer Pro – Таймер продуктивности 
Повышайте свою эффективность с помощью методики Pomodoro прямо в терминале!
Настраивайте интервалы, отслеживайте статистику, получайте уведомления и управляйте временем.
Реализован на 7 языках программирования с единым интерфейсом командной строки.

🚀 Возможности
Таймер Pomodoro – работа (25 мин), короткий перерыв (5 мин), длинный перерыв (15 мин).

Управление – старт, пауза, возобновление, остановка, сброс.

Настройка интервалов – изменение времени работы и перерывов.

Статистика – количество завершённых помидоров, общее время работы, количество перерывов.

Сохранение данных – статистика и настройки хранятся в файле.

Звуковые уведомления – сигнал по окончании интервала.

Интерактивный режим – команды в реальном времени.

Цветной вывод – наглядное отображение состояния.

Кроссплатформенность – Windows, Linux, macOS.

📖 Использование
Синтаксис (единый для всех версий):

bash
<команда> <действие> [аргументы]
Действия
Команда	Описание
start	Запустить таймер (или возобновить после паузы)
pause	Приостановить таймер
resume	Возобновить таймер после паузы
stop	Остановить текущий интервал (сброс)
status	Показать текущее состояние (оставшееся время, фаза)
set work <мин>	Установить время работы (минуты)
set break <мин>	Установить время короткого перерыва
set longbreak <мин>	Установить время длинного перерыва
set cycles <N>	Установить количество циклов до длинного перерыва
stats	Показать статистику
reset	Сбросить статистику
quit	Выход
Примеры
bash
# Запустить таймер
pomodoro start

# Приостановить
pomodoro pause

# Возобновить
pomodoro resume

# Установить время работы 30 минут
pomodoro set work 30

# Показать статистику
pomodoro stats

🧠 Логика работы
Таймер – отсчитывает время в секундах, обновляет состояние каждую секунду.

Фазы – работа (Work), короткий перерыв (Break), длинный перерыв (LongBreak).

Циклы – после N циклов работы следует длинный перерыв.

Статистика – сохраняется в JSON-файле в домашней папке.

Звук – при завершении интервала воспроизводится звуковой сигнал (системный звонок).

✨ Дополнительные фичи
Цветной вывод – текущее состояние выделено цветами.

Обновление строки – оставшееся время отображается в одной строке.

Автосохранение – настройки и статистика сохраняются автоматически.

Кроссплатформенность – все версии работают одинаково.
